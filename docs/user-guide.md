## 首次登录

### 1. 访问系统

打开浏览器，访问部署地址：
- 本地部署：`http://localhost:3000`
- K8s 部署：`https://wiki.yourdomain.com`

### 2. 默认账号

| 字段 | 值 |
|------|-----|
| 邮箱 | `admin@routin.ai` |
| 密码 | `Admin@123` |

**⚠️ 重要**：首次登录后请立即修改密码！

### 3. 修改密码

1. 点击右上角头像
2. 选择「个人设置」
3. 点击「修改密码」
4. 输入新密码并保存

## 创建知识库

### 步骤 1：新建仓库

1. 点击左侧「+ 新建仓库」按钮
2. 填写仓库信息：
   - **仓库名称**：显示名称（如 "MyProject Wiki"）
   - **仓库标识**：URL 标识（如 "myproject"）
   - **描述**：简短描述

### 步骤 2：选择导入方式

| 方式 | 适用场景 |
|------|----------|
| **Git 仓库** | 从 GitHub/GitLab/Gitee 等导入 |
| **ZIP 上传** | 本地项目打包上传 |
| **本地目录** | 直接上传本地文件夹 |

### 步骤 3：配置分析选项

- **启用代码分析**：自动分析代码结构和依赖
- **生成 Mermaid 图**：生成架构图和流程图
- **语言**：选择文档生成语言（中文/英文）
- **排除目录**：设置不需要分析的目录（如 `node_modules`, `.git`）

### 步骤 4：开始导入

点击「开始导入」，系统将：
1. 拉取/上传代码
2. 分析代码结构
3. 生成文档目录
4. 生成内容

**⚠️ 注意**：大型仓库可能需要较长时间，请耐心等待。

## 导入代码仓库

### 从 GitHub 导入

1. 获取仓库 URL：`https://github.com/username/repo.git`
2. 选择「Git 仓库」导入方式
3. 粘贴仓库 URL
4. 如为私有仓库，填写访问令牌（Token）
5. 选择分支（默认 `main` 或 `master`）

### 从 GitLab 导入

与 GitHub 类似，使用 GitLab 仓库 URL：
- `https://gitlab.com/username/repo.git`
- 或自托管 GitLab：`https://gitlab.company.com/group/repo.git`

### 使用访问令牌

对于私有仓库，需要创建访问令牌：

**GitHub**：
1. 进入 Settings → Developer settings → Personal access tokens
2. 创建 Token，勾选 `repo` 权限
3. 复制 Token 到导入页面

**GitLab**：
1. 进入 User Settings → Access Tokens
2. 创建 Token，勾选 `read_repository` 权限

### ZIP 文件上传

1. 将项目文件夹打包为 `.zip` 文件
2. 选择「ZIP 上传」方式
3. 拖放或选择文件
4. 等待上传完成

**建议**：ZIP 文件大小不超过 500MB

## AI 对话功能

### 启动 AI 对话

1. 打开任意文档
2. 点击右侧「AI 对话」按钮
3. 输入问题开始对话

### 常用提问示例

**了解代码功能**：
```
这个函数的主要作用是什么？
请解释这段代码的业务逻辑。
```

**代码改进建议**：
```
这段代码有什么可以优化的地方？
如何提高这个函数的性能？
```

**生成文档**：
```
请为这个模块生成 API 文档。
生成这段代码的使用示例。
```

**排查问题**：
```
这段代码可能存在什么 Bug？
解释一下这个错误的原因。
```

### 对话历史

- 系统自动保存最近 50 条对话
- 点击「历史记录」查看
- 可以收藏重要对话

## 文档生成

### 自动生成文档

系统会自动生成以下文档：

| 文档类型 | 说明 |
|----------|------|
| **项目概述** | 项目简介、技术栈、目录结构 |
| **架构图** | Mermaid 系统架构图 |
| **API 文档** | 接口定义、参数说明 |
| **更新日志** | 基于 Git 提交生成 |

### 手动触发生成

1. 进入知识库设置
2. 点击「重新生成文档」
3. 选择生成范围：
   - 全部文档
   - 仅更新部分

### 增量更新

系统支持自动增量更新：
1. 进入仓库设置
2. 开启「自动同步」
3. 设置同步间隔（如每小时）

## 常见问题

### Q: 导入的仓库显示「分析中」很久？

A: 大型仓库可能需要 10-30 分钟。可以点击「查看日志」了解进度。

### Q: AI 回答不够准确？

A: 尝试：
1. 提供更具体的上下文
2. 指定具体的文件或函数名
3. 检查 AI API Key 是否配置正确

### Q: 如何导出生成的文档？

A: 进入知识库 → 设置 → 导出，选择格式（Markdown/PDF/HTML）。
